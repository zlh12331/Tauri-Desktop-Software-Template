## 入门

欢迎！本指南涵盖应用中可用的核心功能。

## 键盘快捷键

### 全局快捷键

| 快捷键   | Mac    | Windows/Linux | 操作             |
| -------- | ------ | ------------- | ---------------- |
| 命令面板 | Cmd+K  | Ctrl+K        | 打开命令面板     |
| 偏好设置 | Cmd+,  | Ctrl+,        | 打开偏好设置     |
| 快捷面板 | 可配置 | 可配置        | 打开快捷输入面板 |
| 左侧栏   | Cmd+1  | Ctrl+1        | 切换左侧栏       |
| 右侧栏   | Cmd+2  | Ctrl+2        | 切换右侧栏       |

## 核心功能

### 命令面板

按 **Cmd+K** 打开命令面板 - 一种快速查找和执行任何操作的方式。开始输入即可搜索可用命令。

### 快捷面板

快捷面板是一个小型浮动窗口，可以通过全局键盘快捷键调出，即使应用在后台运行时也可以使用。使用它进行快速数据输入或执行操作，无需切换到主窗口。

在 **偏好设置 → 键盘快捷键** 中配置快捷面板的快捷键。

### 偏好设置

按 **Cmd+,** 打开偏好设置：

- **主题**：浅色、深色或跟随系统
- **语言**：选择您偏好的语言
- **键盘快捷键**：自定义快捷面板的快捷键

### 原生菜单

从菜单栏访问各项功能：

- **应用菜单**：关于、检查更新、偏好设置、退出
- **视图菜单**：切换侧边栏

所有菜单项都有键盘快捷键，同时也可以在命令面板中使用。

## 布局

- **标题栏**：窗口控件和应用标题
- **左侧栏**：可折叠面板（Cmd+1）
- **主内容区**：主要应用内容
- **右侧栏**：可折叠面板（Cmd+2）

## 更新

应用会自动检查更新：

- 手动检查：应用菜单 → 检查更新
- 更新从 GitHub releases 下载
- 有可用更新时您将收到通知
