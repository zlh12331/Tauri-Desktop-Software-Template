# Security Policy

## Reporting a Vulnerability

Do not report security vulnerabilities through public GitHub issues.

Instead, please [open a private security advisory on GitHub](https://github.com/zlh12331/Tauri-Desktop-Software-Template/security/advisories/new).

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fixes (if any)

### Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial Assessment**: Within 1 week
- **Resolution**: Depends on severity, typically 2-4 weeks for critical issues

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| < 1.0   | ❌        |
